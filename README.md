## âš™ï¸ Installation & Setup

To run this project locally, follow these steps:

### ðŸ”§ Backend (Node.js)

```bash
# 1. Clone the repository
git clone https://github.com/abdullaabdulla97/portfolio-project.git
cd portfolio-project/Backend

# 2. Install dependencies
npm install

# 3. Create a .env file to add the Resend API key
echo "RESEND_API_KEY=your_resend_api_key_here" > .env

# 4. Start backend server
node server.js
```
### ðŸŽ¨ Frontend (React)

```bash
# 1. Navigate to frontend
cd ../Frontend

# 2. Install dependencies
npm install

# 3. Create a .env file to add the localhost URL
echo "REACT_APP_API_URL=http://localhost:5000" > .env

# 4. Start the development server
npm start
```
